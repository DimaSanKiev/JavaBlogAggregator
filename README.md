<h2>Java Blog Aggregator</h2>

<h4>Overview</h4>
This application can download feeds from specified URLs and store them inside database. On the home page there are the latest news from all feeds from all blogs.<br>
Anyone can register. jQuery is used for client-side validation. After registration user can sign in and go to his account, where they can add some new blog. The news from this blog will be immediately visible on the home page.<br>
Administrator can see all users, their blogs and can remove users and their blogs.

<h4>Technologies used</h4>
<b>Hibernate</b>, <b>Spring</b> (basic framework, <b>Spring Data JPS</b>, <b>Spring Security</b>), <b>Twitter Bootsrap</b>, <b>jQuery</b>, <b>Apache Tiles</b>, <b>JSP</b>.<br>
<b>Jetty</b> server is used for testing.
